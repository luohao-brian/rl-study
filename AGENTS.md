# 强化学习智能体 (Agents)

本文档用于记录和说明项目中实现的强化学习智能体。目前项目包含两种经典强化学习算法的实现：蒙特卡洛（MC）智能体和深度 Q 网络（DQN）智能体。

## 1. 蒙特卡洛智能体 (MC Agent)
- **文件位置**: `maze/mc_agent.py`
- **核心思想**: 基于完整回合（Episode）的经验进行学习，通过采样多条轨迹，计算每个状态-动作对的平均累计回报来更新 Q 值。
- **适用场景**: 适用于回合制的离散环境，且状态空间相对较小的问题。

## 2. 深度 Q 网络智能体 (DQN Agent)
- **文件位置**: `maze/dqn_agent.py`
- **核心思想**: 结合了 Q-Learning 与深度神经网络。使用神经网络来逼近 Q 值函数，能够处理状态空间巨大的复杂问题。通过经验回放（Replay Buffer）打破数据相关性，提升训练稳定性。
- **适用场景**: 适用于状态空间连续或极其庞大的复杂强化学习问题。
