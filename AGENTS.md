# 强化学习智能体 (Agents)

本文档用于记录和说明项目中实现的强化学习智能体。目前项目涵盖了强化学习从经典表格型到深度网络型的核心基础算法：动态规划（DP）、蒙特卡洛（MC）、时间差分（TD）以及深度 Q 网络（DQN）。

## 1. 动态规划智能体 (DP Agent)
- **文件位置**: `maze/dp_agent.py`
- **核心思想**: 基于环境的完整动态模型（Model-based），与基于物理试错的 Model-free 范式有着本质区分。智能体利用环境先验特权遍历静态坐标集 `self.env.states`，将底层 `env.step(s, a)` 视为无副作用的确定性转移查询接口（纯函数式提问器）离线查询所有后继状态和期望回报。通过反复执行 Bellman 最优方程的全状态自举扫描（Bootstrapping），计算出最优的状态价值函数 $V(s)$，进而推导出全局最优策略。本模块实现了经典的**价值迭代（Value Iteration）**算法。
- **适用场景**: 适用于环境转移概率和奖励完全已知、且状态空间有限的离散环境规划问题。

## 2. 蒙特卡洛智能体 (MC Agent)
- **文件位置**: `maze/mc_agent.py`
- **核心思想**: 基于完整回合（Episode）的经验进行学习（Model-free），通过采样多条轨迹，事后倒推计算每个状态-动作对的平均累计回报来更新 Q 值。无自举偏差，但方差较大。
- **适用场景**: 适用于具备明确终止状态的回合制离散环境，且状态空间相对较小的问题。

## 3. 时间差分智能体 (TD Agent)
- **文件位置**: `maze/td_agent.py`
- **核心思想**: 结合了 MC 的在线采样与 DP 的单步自举优势。无需等待回合结束，每走一步即可利用下一状态的价值估计当场更新当前 Q 值。模块内部集成了两大主流控制算法：
  - **Q-Learning**（异策略 Off-policy）：直接以最大未来 Q 估值作为自举目标，勇于探索最优上界。
  - **Sarsa**（同策略 On-policy）：严格遵循当前探索策略预采样下一动作计算期望，学习行为更加稳健安全。
- **适用场景**: 适用于无法获取环境全局模型、支持单步持续交互或长周期运行的强化学习控制任务。

## 4. 深度 Q 网络智能体 (DQN Agent)
- **文件位置**: `maze/dqn_agent.py`
- **核心思想**: 结合了 Q-Learning 与深度神经网络。使用神经网络来逼近 Q 值函数，能够处理状态空间巨大的复杂问题。通过经验回放（Replay Buffer）打破数据相关性，提升训练稳定性。
- **适用场景**: 适用于状态空间连续或极其庞大的复杂强化学习问题。
